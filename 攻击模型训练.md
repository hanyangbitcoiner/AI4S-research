## 如何启动训练

> [!IMPORTANT]
> 在同一台机器上，建议将 GPU 进行分配。例如：使用卡 0 跑 vLLM，卡 1-7 跑训练。

### 步骤 1: 启动 vLLM 服务
在一号终端运行（使用卡 0）：
```bash
cd TrainLLMs3
bash scripts/launch_vllm.sh "Qwen/Qwen2.5-7B-Instruct" "0" 8000
```

### 步骤 2: 验证连接
运行测试脚本确保 API 可用：
```bash
python TrainLLMs3/tests/test_vllm_connection.py --url http://localhost:8000/v1
```

### 步骤 3: 启动分布式训练
在二号终端运行（使用卡 1-7）：
```bash
cd TrainLLMs3
export CUDA_VISIBLE_DEVICES="1,2,3,4,5,6,7"
bash scripts/launch_rl_training_distributed.sh
```
